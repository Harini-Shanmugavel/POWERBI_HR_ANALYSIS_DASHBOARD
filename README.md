**HR Analytics Dashboard (Power BI)**

 **Project Overview**
 
This project focuses on analyzing HR data to gain insights into employee performance, attrition, and workforce trends.
The dashboard helps HR teams make data-driven decisions by visualizing key metrics.

**Objectives**

Analyze employee attrition and retention patterns
Understand workforce demographics
Identify factors affecting employee performance
Provide actionable insights for HR decision-making

**Tools & Technologies**

Power BI
Excel (Data Source)
DAX (Data Analysis Expressions)
